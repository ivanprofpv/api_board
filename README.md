## Board_api на FastAPI

Асинхронное API для доски объявлений

## Что можно сделать в приложении:

- Запросить карточку;
- Добавить карточку;
- Отредактировать ее;
- И удалить;
- Создать категории объявлений;
- Отдать карточки по категориям.
